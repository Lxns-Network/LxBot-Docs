# 象棋
<span class="span-group">群聊</span>

---

## 创建房间
```
/cc create
```

## 加入房间
```
/cc join <房间 ID>
```
`<房间 ID>` 由[创建房间](#创建房间)生成。

## 删除房间
> 您需要在一个房间中。
```
/cc stop
```

## 获取棋盘
该指令用于图片无法正常获取时的重试方案。
```
/cc get
```

## 移动棋子
> 您需要在一个房间中。
```
/cc move <位置>
```
`<位置>` 如 a3a4。

## 清空所有房间
<span class="span-bot-admin">Bot 管理员</span>
```
/cc clear
```
