## 服务器

### 切换默认语言
切换将会实时生效，但仅针对部分已翻译的模块。
```
/dc server lang [语言]
```
`[语言]` 为空时返回当前语言，可选值：
- 简体中文（默认）：**zh**
- 繁體中文：**tw**
- English：**en**
- 日本語：**jp**
- 한국어：**kr**
- Français：**fr**

