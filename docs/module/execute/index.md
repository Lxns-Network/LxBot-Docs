## 运行
```
/e <语言> <代码>
```
`<语言>` 可选值：
- Python 3：`py`
- Node.js：`js`
- Lua：`lua`
- C++：`cpp`
- C#：`cs`
- C：`c`
- Java：`java`
- PHP：`php`
- Kotlin：`kt`
- Kotlin Script：`kts`
- Rust：`rs`
- Golang：`go`
- F#：`fs`
- R：`r`
- Pascal：`pas`
- Lisp：`lisp`
- Erlang：`erl`
- Swift：`swift`
- Brainfuck：`bf`
- Groovy：`groovy`
- Bash：`sh`
- Ruby（暂不可用）：`rb`
- Scala（暂不可用）：`scala`
