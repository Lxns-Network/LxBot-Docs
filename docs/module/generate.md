## 图片生成
```
/gen <样式> [@]
```
`<样式>` 可选值（GIF 样式）：
- 蹭：`ceng`
- 吃：`chi`
- 冲：`chong`
- 搓：`cuo`
- 顶：`ding`
- 丢：`diu`
- 啃：`ken`
- 摸：`mo`
- 拍：`pai`
- 敲：`qiao`

`[@]` 为空时以自己头像作为素材，反之以目标头像作为素材。
