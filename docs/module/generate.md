## 图片生成
```
/gen <样式> [目标或其它参数]
```
`<样式>` 可选值：
- GIF 样式：
  - 蹭：`ceng`
  - 吃：`chi`
  - 冲：`chong`
  - 锤：`chui`
  - 锤2：`chui2`
  - 搓：`cuo`
  - 捣：`dao`
  - 顶：`ding`
  - 丢：`diu`
  - 紧贴：`jintie`
  - 啃：`ken`
  - 摸：`mo`
  - 膜拜：`mobai`
  - 拍：`pai`
  - 拍2：`pai2`
  - 敲：`qiao`
  - 敲2：`qiao2`
  - 嗦：`suo`
  - 永远爱你：`yyan`
- JPG 样式：
  - 喜报：`xibao <文本>`
  - 悲报：`beibao <文本>`
  - 小天使：`xts [目标 [名称]]`

`[目标]` 为空时以自己头像作为素材，反之以目标头像作为素材。
