## 查询服务器详情
``` {1}
/mc ping <服务器 IP[:端口]>
/minecraft ping <服务器 IP[:端口]>
```
`[:端口]` 默认为 **:25565**。

## 获取玩家皮肤
``` {1}
/mc skin <用户名>
/minecraft skin <用户名>
```
`<用户名>` 仅限正版 ID。
