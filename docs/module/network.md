## Ping
```
/n ping <主机名> [次数]
```
`[次数]` 为整数，最大为 10 次。

## DNS 查询
```
/n dns <主机名> <记录名>
```
`<记录名>` 包含 **A**, **AAAA**, **CNAME**, **MX**, **NAPTR**, **NS**, **PTR**, **SOA**, **SRV**, **TXT**。
