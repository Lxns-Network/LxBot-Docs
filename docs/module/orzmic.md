## 曲目详情
```
/orz song <曲名>
```

## 定数计算
```
/orz calc <分数> <曲名 [难度]>
```
`[难度]` 可选值：
- EASY：`ez` 或 `easy`
- NORMAL：`nm` 或 `normal`
- HARD（默认）：`hd` 或 `hard`
- SPECIAL：`sp` 或 `special`
