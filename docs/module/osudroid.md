```
/osud bind <osu!droid 用户名>
/osudroid bind <osu!droid 用户名>
```

## 查询单曲最佳成绩
```
/od info <SID>
/osud info <SID>
/osudroid info <SID>
```
`<SID>` 为 Beatmapset ID。
