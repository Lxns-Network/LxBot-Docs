---

## 绑定账号
```
/od bind <osu!droid 用户名>
```

## 查询最近成绩
```
/od [info [@]]
```
`[@]` 为空时默认自己账号，反之为目标账号。

## 查询单曲最佳成绩
```
/od info <SID>
```
`<SID>` 为 Beatmapset ID。

## 查询多次成绩
获取最近 5 次的成绩。
```
/od multi
```

## CDPP 详情
```
/od cdpp
```

## CDPP 上传成绩
```
/od update [SID]
```
`[SID]` 为 Beatmapset ID。
