## RKS 计算
``` {1}
/pgr calc <曲名 [难度]|定数> <ACC%>
/pgr calculate <曲名 [难度]|定数> <ACC%>
/phi calc <曲名 [难度]|定数> <ACC%>
/phi calculate <曲名 [难度]|定数> <ACC%>
/phigros calc <曲名 [难度]|定数> <ACC%>
/phigros calculate <曲名 [难度]|定数> <ACC%>
```
`[难度]` 可选值：
- EZ：`ez`
- HD：`hd`
- IN（默认）：`in`
- AT：`at`
- Legacy：`legacy`
