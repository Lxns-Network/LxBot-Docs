## 生成普通样式
```
/qr <文本>
/qr gen <文本>
/qr generate <文本>
/qrc <文本>
/qrc gen <文本>
/qrc generate <文本>
/qrcode <文本>
/qrcode gen <文本>
/qrcode generate <文本>
```

## 生成图片背景
```
/qr img <文本>
/qr image <文本>
/qrc img <文本>
/qrc image <文本>
/qrcode img <文本>
/qrcode image <文本>
```
`<文本>` 不支持中文等字符。

随后单独发送 `<图片>`，发送其它文本则取消操作。
