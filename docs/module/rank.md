# 发言排行
<span class="span-group">群聊</span>

---

## 群成员发言
``` {1}
/rank
/rank this
/rank this [范围 [日期]]
```
`[范围]` 可选值：
- 本日（默认）：`1d`
- 本周：`1w`
- 本月：`1m`

`[日期]` 格式为 **yyMMdd**，如 `201108`。

## 成员总发言
```
/rank all [范围 [日期]]
```
`[范围]` 可选值：
- 本日（默认）：`1d`
- 本周：`1w`
- 本月：`1m`

`[日期]` 格式为 **yyMMdd**，如 `201108`。
